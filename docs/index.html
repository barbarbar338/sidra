⚔️ SidraJS

Sidra offers you many decorators for you to use with the express module.

TypeScript definitions are built in.

Installation
  • With CLI:
    • Run
      $ npm install --global --save sidra
      or
      $ yarn global add sidra
    • Run
      $ sidra create
  • Without CLI:
    • Run
      $ npm install --global --save sidra
      or
      $ yarn global add sidra
